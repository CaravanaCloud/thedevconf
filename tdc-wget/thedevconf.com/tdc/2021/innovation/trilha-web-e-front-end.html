Trilha WEB / FRONT-END

Quer aprender mais sobre desenvolvimento Web? Inscreva-se na trilha que vamos te ajudar a revolucionar o futuro da tecnologia.

Quando falamos de desenvolvimento web isso envolve varios temas em desenvolvimento de software que nos conecta com diverssas áreas. O mesmo ocorre quando falamos de front-end, onde limitamos um pouco o campo de estudo e atuação para algo bastante visual e intuitivo, as interfaces de sistemas e sites que desenvolvemos são projetadas para passar uma menssagem para o usuário de forma que consigam executar as ações necessárias.

Na nossa trilha vamos compartilhar conhecimento do nosso dia-dia de desenvolvimento, e apresentar tecnologias, metodologias ou ferramentas que vocês podem se aprofundar nos temas no futuro.

Revitalising the Web - The importance of Web Performance

Tamas Piros

With the growth of the web, websites have grown as well. Unfortunately, this growth is not always a positive one, especially when it comes to load times. In today?s web ecosystem, sites are more interactive and visual than ever. This has certain drawbacks and can ultimately lead decreased user engagement or even loss of revenue. During this talk we?ll discover the most important aspects of web performance, focussing on today?s modern web architectures and measurement tools. By the end of the talk you?ll be able to interpret lighthouse and page insights results while also be able to understand the core web vital metrics.

Pare de testar seu código frontend!

Bruno Vercelino da Hora

Isso mesmo! Pare de testar seu código frontend! Calma? calma... abaixe as pedras! Você que é amante de teste unitário (e eu sou um deles), não precisa me apedrejar. Como você desenvolve seus testes unitários? Você testa cada detalhe de suas funções e classes? Você testa cada detalhe de implementação? Pois bem, é isso que você deve parar de fazer. Ao testar uma aplicação frontend você deve focar em criar testes que reproduzem o comportamento do usuário. Quero mostrar nesta talk porque testar os detalhes de implementação é uma péssima abordagem e mostrar como podemos criar testes livres de detalhes de implementação usando a biblioteca testing-library disponível para React, Vue e Angular.
